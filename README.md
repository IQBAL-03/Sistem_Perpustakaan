Sistem Informasi Perpustakaan adalah aplikasi berbasis web untuk mengelola **data buku, pengguna, peminjaman, dan pengembalian buku** secara efisien dan terstruktur.

Project ini dibuat menggunakan **PHP Native dan MySQL**.

## Fitur Utama

* Login & manajemen pengguna (Admin / Staff)
* CRUD data buku
* Transaksi peminjaman & pengembalian buku
* Update stok buku otomatis
* Laporan stok buku real-time
* Dashboard admin & staff

## Keamanan

* Password di-hash menggunakan **bcrypt**
* Validasi form server-side
* Prepared Statement (Anti SQL Injection)
* Session login

## Teknologi

* PHP Native
* MySQL
* Bootstrap 5
* JavaScript (AJAX)
